SISTEMA DE GESTION BASICA DE ESTUDIANTES

•	Ingreso datos de estudiante (Nombre, edad, 3 notas)

•	Calculo de promedio de las notas

•	Se determina el estado de estudiante (Aprobado, En recuperación, Reprobado)

•	Se puede ingresar varios datos de estudiantes a través del menú

•	Se muestra al final un promedio general de los estudiantes ingresados
